# decision-paralysis:

## This project is a smart activity suggestion tool that instantly:

 - Shows only highly-rated places
 - Fits within the user’s budget
 - Matches the available time
 - Combines multiple activities automatically (ex: park + café)
 - Creates a ready-to-go mini plan
Now users can decide in seconds instead of minutes.

## Example:

User enters:
Budget: ₹300
Time: 1 hour

## App suggests:

 -🌳 Park (30 mins – Free)
 -☕ Café (30 mins – ₹200)
 -Total: ₹200 | 1 hour ✔️

## Key Features:

 - Smart suggestion button
 - Budget-based filtering
 - Time-based planning

## Nearby places (Google Maps):

 - Simple and fast UI
 - Goal

To help people stop overthinking and start enjoying quick, spontaneous outings.
